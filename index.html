How to enter values for a given quadratic equation?

The coefficients a, b and c for a given quadratic equation (polynomial of degree 2) can be obtained by first writing the quadratic equation in standard form, \( \; ax^2 + bx +c = 0, \;\; a \ne 0\).
The values of b and c can be 0 but if a equals 0, the equation will become linear.

Example 1

\( 2x^2 - x + 5 = 0\)
For this given quadratic equation the values of a, b and c are $$ \bbox[4pt,border: 1px solid grey]{a = 2, \; b = -1, \; c = 5} $$

Example 2

\( x^2 - 16 = 0\)
The value of b is 0 for this quadratic equation. $$ \bbox[4pt,border: 1px solid grey]{a = 1, \; b = 0, \; c = -16} $$

Example 3

\( (x - 3)(x + 5) = -3\)
At first glance the given equation does not look to be a quadratic equation. Expand the brackets and collect all the terms at the left side of the equation, $$ x^2 + 5x -3x - 15 +3 = 0 $$ $$ x^2 +2x -12 = 0 $$ The equation turned out to be a quadratic with values of a, b and c as, $$\bbox[4pt,border: 1px solid grey]{ a = 1, \; b = 2, \; c = -12} $$

Example 4

\( (3x - 2) - (x - 3)^2 + 11 = 0\)
Use formula from algebra, \( (a - b)^2 = a^2 - 2ab + b^2\), to expand the bracket in the given equation and collect all the like terms, $$ 3x - 2 - (x^2 - 2 . x . 3 + 9) + 11 = 0 $$ $$ -x^2 +6x +3x -9 -2 +11 = 0 $$ $$ -x^2 + 9x = 0 $$ $$ x^2 - 9x = 0 $$ The values of a, b and c are, $$\bbox[4pt,border: 1px solid grey]{ a = 1, \; b = -9, \; c = 0} $$

Example 5

$$ \frac{x+3}{x-3} + \frac{x-6}{x+6} = \frac{11}{5} $$
Multiply both side of the equation by \( 5(x-3)(x+6) \) to simplify the fractions. $$ \frac{x+3}{x-3} (5)(x-3)(x+6) + \frac{x-6}{x+6} (5)(x-3)(x+6) = \frac{11}{5} (5)(x-3)(x+6) $$ $$ 5(x+3)(x+6) + 5(x-3)(x-6) = 11(x-3)(x+6) $$ Expand brackets and collect all terms on the left side of the equation, $$ 5(x^2 +3x +6x + 18) + 5(x^2 -3x -6x + 18) = 11(x^2 -3x +6x -18)$$ $$ 5(x^2 +9x + 18) + 5(x^2 -9x + 18) = 11(x^2 +3x -18)$$ $$ 5x^2 +45x + 90 + 5x^2 -45x + 90 = 11x^2 +33x - 198$$ $$ 10x^2 +180 = 11x^2 +33x - 198$$ $$ 10x^2 -11x^2 -33x +180 + 198 = 0$$ $$ -1x^2 -33x + 378 = 0$$ $$ x^2 +33x - 378 = 0$$ The values of a, b and c are, $$\bbox[4pt,border: 1px solid grey]{ a = 1, \; b = 33, \; c = -378} $$
